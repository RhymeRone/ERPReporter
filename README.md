
# Finansal Raporlama Yazılımı
Bu yazılım, Nebim v3 Ön Muhasebe Programı ile entegre çalışarak işletme yönetiminde kullanıcıya daha pratik ve sadeleştirilmiş raporlar sunmayı amaçlamaktadır. 





## Özellikler

- Kullanıcı giriş paneli ( Beni Unutma özelliği )
- Farklı sunuculara / veritabanlarına erişim imkanı
- Bağlantı testi
- Personelleri / Şubeleri / Müşterileri / Satışları Listeleme
- Kasa Bilgisi
- Çek / Senet Bilgisi
- Kâr ve Zarar Bilgisi
- Ciro Raporu
- Gelir ve Gider 
- Program Hakkında Bilgi 
- Güncelleme Kontrolü / Otomatik Güncelleme Desteği
- Gelişmiş Rapor Paneli ( PDF / Mail / Ekran Görüntüleri / Çoklu Rapor Açma / Veri Sayısı ve Toplamı / Gelişmiş Filtreleme Özellikleri )


#### Geliştirmeye açık özellikler
- Personel Maaşları / Personel Gider Tablosu / Personel Satış Raporu
- Gecikmede ki Müşteriler
- İade Alınan Müşteriler
- En Çok / En Az Satış Yapılan Müşteriler
- En Çok / En Az Satılan Ürünler
- Ürün Stok Durumu / Olası Kazanç Durumu / Kâr ve Zarar Tablosu
.

  
## Kullanılan Teknolojiler

- DevExpress v22.1
- Windows Forms
- T-SQL 
- C# 
- Visual Studio
- Auto Updater .NET
